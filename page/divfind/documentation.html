Preamble

Welcome to DivFind documentation. In this page I describe how DivFind works.

Doing a search

You can do a search in Firefox by clicking on the widget. Also you can select the search terms and do a right click. On Fennec you need to do a long click. On Thunderbird and SeaMonkey do a right click. There's also an option to search from a protocol bu for activate this you need first to open DivFind and allow this. This URL shows a search when this is enabled: web+divfind:{searchTerms}

Installing new search engines

There are two types of search engines. First, Firefox native search engines (search engines avalibles in the second top bar). Second, DivFind search engines. For install and delete the first search engines you need to go to the search manager in Firefox. For DivFind search engines go to "Manage" and go to "ADD" or "DELETE". For adding a search engine you have to complete these parameters Name: The name of the search engine
Description: A little description about the search engine
Favicon URL: A URL to the search engine icon
URL: A URL with a GET parameter where DivFind puts the search terms. Example: http://developer.mozilla.org/es/search?q={searchTerms}

Some problems

Some servers blocks the iframe with a Server parameter. These search engines are displayed in white. Some examples of this servers are: Google, Twitter, MDN, etc
