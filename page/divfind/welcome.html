DivFind

Description

DivFind is a metasearch engine wich takes your search engines and the native Firefox engines (if avalible) to see all the results of search in one page

Collaborate

DivFind is open-source under the GPL 2.0. You can collaborate:

  • Writing a review
  • Doing pull requests
  • Sharing DivFind
