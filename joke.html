Firefox changes Gecko,the engine, for Trident, the engine used in Internet Explorer

CEO at Mozilla says: “Trident is the engine with less bugs. We will use Trident for IE6”

Today, the CEO at Mozilla says the new and all the online newspapers collect her phrases:

  • “Trident is the engine with less bugs in the history of computation”
  • “JavaScript? Too fast. We are going to use Visual Basic Script and of course, ActiveX controls”
  • “Linux, Mac OS X, Solaris, FreeBSD, Windows 7, ... These systems are for bored people. Cool people use Windows, of course, Windows XP SP3”
  • “Addons? NO, of course not. The new Firefox generation will come with a interface designed in Visual Basic 6.”
  • “IRC Client? IRC is from the past. We're going to use the most advanced comunicated system in the world: Free Hotmail and MSN Messenger”
  • WebGL is almost good but the game industry is preocupated so we're going to remove it from the new Firefox generation
  • Tabs are from the past. We use Windows

The CEO haven't confirmed yet if this decision will apply other Mozilla products like Thunderbird, SeaMonkey, Sunbird, Fennec and Firefox OS.
