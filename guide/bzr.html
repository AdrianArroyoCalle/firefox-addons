Bazaar guide for firefox-addons

These are quick git commands to use with Firefox-addons

Get a working copy

bzr branch url

Update the code

bzr merge

Do a commit (always do this)

bzr commit -a -m "Message"

Adding files/Removing files/Moving files

bzr add file.ext bzr rm file.ext bzr mv file.ext

See a list of commits

bzr log

Delete a change

bzr reset --hard COMMIT_NAME

Get a patch

bzr diff > mychanges.patch

Apply a patch

bzr apply mychanges.patch

Merge changes

bzr push url branch

Do a branch

bzr branch PATH

Merge a branch

bzr merge PATH

Send changes by email

bzr send
