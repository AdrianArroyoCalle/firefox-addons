Subversion guide

These are some quick-steps in Subversion

Get a working copy

svn checkout https://github.com/AdrianArroyoCalle/firefox-addons --username=USERNAME

Update the code

svn update

Do a commit

svn commit -m "MESSAGE"

Adding files/Removing files/Moving files

svn add FILE svn delete FILE svn move SRC DEST

See a list of changes/commits

svn log

Delete a change

svn revertBefore a commit

Get a patch

svn diff > mypatch.patch

Apply a patch

patch -p0 mypatch.patch

Do a branch

svn copy . BRANCH_PATH -m "MESSAGE"

Change to a branch

svn switch BRANCH-PATH

Merge a branch

svn merge -r REV:HEAD BRANCH_PATH
