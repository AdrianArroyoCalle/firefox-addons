GIT guide for firefox-addons

These are quick git commands to use with Firefox-addons

Get a working copy

git clone git://www.github.com/AdrianArroyoCalle/firefox-addons.git

Update the code

git pull

Do a commit (always do this)

git commit -a -m "Message"

Adding files/Removing files/Moving files

git add file.ext git rm file.ext git mv file.ext

See a list of commits

git log

Delete a change

git reset --hard COMMIT_NAME

Get a patch

git diff > mychanges.patch

Apply a patch

git apply mychanges.patch

Merge changes

git push

Do a branch

git checkout -b BRANCH_NAME

Change to a branch

git checkout BRANCH_NAME

Merge a branch

git merge BRANCH_NAME

Delete an old branch

git branch -d BRANCH_NAME

Get branch list

git branch
