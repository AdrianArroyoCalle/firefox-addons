firefox-addons

Mozilla Firefox, SeaMonkey, Thunderbird and Fennec addons created by Adrián Arroyo Calle

Also there's some addons that also works on Instantbird, BlueGriffon and Nighingale, for this check the description page

You can collaborate to the project by submitting ideas, bugs, spreeding the addons, talking on Social Networks and by coding at GitHub

Every new addon is made under Addon SDK but we can also create UIs for XUL based products like SeaMonkey or Thunderbird via the XUL_UI module, a module developed by us. All new addons needs to be restartless for provide a better user experience than others
